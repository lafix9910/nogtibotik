# Telegram-бот для записи к мастеру маникюра

Полностью готовый к запуску Telegram-бот на Python с использованием библиотеки aiogram 3.x и SQLite.

## Структура проекта

```
project/
├── bot.py (main.py)   # Главный файл запуска бота
├── config.py          # Конфигурация бота
├── database.py        # Модуль базы данных SQLite
├── handlers.py        # Обработчики команд и колбэков
├── keyboards.py       # Клавиатуры (inline и reply)
├── states.py          # FSM состояния
├── utils.py           # Утилиты (проверка подписки, планировщик)
├── requirements.txt   # Зависимости
└── nail_salon.db     # База данных SQLite (создаётся автоматически)
```

## Функционал бота

### Для пользователей:
- 📅 Выбор даты из календаря (1 месяц вперёд)
- ⏰ Выбор доступного временного слота
- 👤 Ввод имени и номера телефона
- ✅ Подтверждение записи
- ❌ Отмена своей записи
- 💰 Просмотр прайсов
- 🖼️ Просмотр портфолио

### Для администратора:
- ➕ Добавление рабочих дней
- ⏰ Добавление/удаление временных слотов
- 🔒 Закрытие/открытие дней
- 📋 Просмотр расписания на дату
- ❌ Отмена записей клиентов

### Дополнительно:
- Проверка подписки на канал
- Автонапоминание за 24 часа до записи
- Уведомление админа о новых записях
- Отправка расписания в Telegram-канал

## Требования

- Python 3.10+
- Telegram-бот (токен)
- Telegram-канал (для проверки подписки и расписания)

## Установка

1. Создайте виртуальное окружение:
```bash
python -m venv venv
```

2. Активируйте виртуальное окружение:
- Windows:
```bash
venv\Scripts\activate
```
- Linux/MacOS:
```bash
source venv/bin/activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка

Отредактируйте файл `config.py`:

```python
# Токен Telegram-бота
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"

# ID администратора (ваш Telegram ID)
ADMIN_ID = 123456789

# ID канала для отправки расписания
SCHEDULE_CHANNEL_ID = -1001234567890

# ID канала для проверки подписки
CHANNEL_ID = -1001234567891

# Ссылка на канал
CHANNEL_LINK = "https://t.me/your_channel"

# Имя мастера
MASTER_NAME = "Мастер маникюра"

# Часовой пояс
TIMEZONE = "Europe/Moscow"
```

## Запуск

```bash
python main.py
```

## Команды

- `/start` - Запуск бота / главное меню
- `/admin` - Админ-панель (только для админа)

## Как получить необходимые ID

1. **BOT_TOKEN**: Создайте бота через @BotFather в Telegram
2. **ADMIN_ID**: Напишите боту @userinfobot, чтобы узнать свой ID
3. **CHANNEL_ID**: Добавьте бота в канал как администратора, используйте @username_to_id_bot для получения ID канала

## База данных

При первом запуске автоматически создаётся база данных SQLite и генерируются рабочие дни на 1 месяц вперёд со стандартными временными слотами (9:00 - 18:00).
